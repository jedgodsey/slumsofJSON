from the slums of JSON

A testing API inspired by the Wu-Tang Clan*

From the Slums of JSON is a real REST API that anyone can use for prototyping or testing. Rather than some gibberish pseudo-latin, it features the words and wisdom of the Wu-Tang Clan.

Yes, unlike the other services that we are shamelessly ripping off, this API is actually real. When you POST, PUT, PATCH or DELETE you are actually changing the databse. If for any reason you'd like to start over with new data, simply refresh or revisit this page in your browser to wipe and re-seed the database.

Resources

From the Slums of JSON currently has the three resources most common to a simple blog and we are working to add more. The resources themselves are related to each other, with each user potentially having several posts and comments and each post potentially having several comments.

Test It Out

Simply fill in the form with a resource and (if necessary) an id number and click "request" to see your response. You'll also see the resulting endpoint used, which you can plug directly into your own application. Want to test out your own API? Simply change the url in the form to you your own and get a response in the same way.
